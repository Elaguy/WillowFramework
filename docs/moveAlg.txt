MOVEMENT ALGORITHM
------------------

Player moves
if tile above/left/right/below player is empty (contains a space)
	normal movement
	
else (tile above/left/right/below player is not empty (contains anything other than a space)
	1. save tile string above/left/right/below player
	2. overwrite player icon on that tile
	3. once player icon is off of that tile, place the saved tile string
	back on that tile
